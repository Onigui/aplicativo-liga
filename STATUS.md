## 🚀 Como Executar o Sistema Completo

### 1. **Backend** (Porta 3001)
```bash
cd backend
npm install
npm start
```

### 2. **Site Principal** (Porta 3000)
```bash
cd public-site
npm install
npm start
```

### 3. **Painel Admin** (Porta 3002)
```bash
cd admin-panel-liga-do-bem
npm install
npm start
```
